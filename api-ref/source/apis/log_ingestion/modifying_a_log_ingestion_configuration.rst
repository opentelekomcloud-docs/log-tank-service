.. _UpdateAccessConfig:

Modifying a Log Ingestion Configuration
=======================================

Function
--------

This API is usde to modify a log ingestion configuration.

URI
---

PUT /v3/{project_id}/lts/access-config

   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

   |                 |                 |                 |                                                                                                                                                                                |
