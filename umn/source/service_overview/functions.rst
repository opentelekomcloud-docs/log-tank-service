Functions
=========

Before using LTS, learn its main functions in :ref:`Table 1 <lts_01_0004__en-us_topic_0180792001_table16113624193111>`.

.. _lts_01_0004__en-us_topic_0180792001_table16113624193111:

.. table:: **Table 1** Functions

   +-----------------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Function                          | Description                                                                                                                                                                                                                                                                                                                                                                        |
   +===================================+====================================================================================================================================================================================================================================================================================================================================================================================+
   | Real-time log collection          | Collects real-time logs and displays them on the LTS console in an intuitive and orderly manner. You can query logs or transfer logs for long-term storage.                                                                                                                                                                                                                        |
   |                                   |                                                                                                                                                                                                                                                                                                                                                                                    |
   |                                   | Collected logs can be structured or unstructured. Log structuring processes logs in log streams by extracting the logs in a fixed format or with a similar pattern based on the extraction rules you set.                                                                                                                                                                          |
   +-----------------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | High-volume storage and search    | Supports quick log query by keyword or fuzzy match.                                                                                                                                                                                                                                                                                                                                |
   +-----------------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Log transfer                      | You can customize the retention period for the host and cloud service logs reported to LTS. Logs older than this period are automatically deleted. For long-term storage (log persistence), you can transfer logs to OBS. Log transfer is to replicate logs to the target cloud service. The original logs remain in LTS and are not deleted until their retention period expires. |
   +-----------------------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
